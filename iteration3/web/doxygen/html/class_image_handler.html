Detailed Description

The ImageHandler class hides all of the implementation details of loading and saving images from the rest of the code.

To load an image using ImageHanlder, you simply call loadImage() with the appropriate image filename and the PixelBuffer to load the image to. To save an image, you call saveImage() with the filename you'd like to save the image to and the PixelBuffer to retrieve the image from. This class supports saving and loading of JPEG images and PNG images with transparency.
