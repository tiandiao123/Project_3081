Detailed Description

A Filter implementation that does not require knowledge of where each modified pixel is located in the image.

PositionIndependentFilters apply a simple function to each ColorData (pixel) object of a PixelBuffer independently. Inner functions included in the library are located in PixelFilterAlgorithm
