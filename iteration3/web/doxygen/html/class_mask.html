Detailed Description

A parent/wrapper class for masks.

Holds a 2D array of floating point values from 0..1. Values often represent opacity of the item being passed through the mask. 0 means none gets through, 1 means it all does. Handles memory allocation and destruction and dynamic sizing of array. Simplifies mask operations by always assuming a square
